나도 바나나 웹서비스 테스트

Flask 서버가 http://localhost:5000 에서 실행 중입니다.

웹서비스 열기

테스트 방법:

  1. 위 링크를 클릭하여 웹서비스를 엽니다
  2. 이미지 파일을 드래그 앤 드롭으로 업로드합니다
  3. 텍스트 프롬프트를 입력합니다 (예: "바나나를 들고 있는 귀여운 캐릭터를 그려줘")
  4. "이미지 생성하기" 버튼을 클릭합니다

주요 기능:

  • ✅ 드래그 앤 드롭 이미지 업로드
  • ✅ 여러 이미지 동시 업로드
  • ✅ 이미지 미리보기
  • ✅ 텍스트 프롬프트 입력
  • ✅ 생성된 이미지 다운로드
