여러 장의 이미지와 텍스트 프롬프트 입력을 받아서 새로운 이미지를 생성해 주는 파이썬 기반 웹서비스를 만들 거야.

주요 기능은 아래와 같아.
1. 이미지파일 불러오기 (여러 개 파일을 한번에 불러올 수 있어야 함, 선택 입력)
   여려개의 파이을  Drag and Drop으로 입력할수 있게 해줘. 
2. 텍스트 프롬프트 입력 (필수 입력)
3. 불러온 이미지 파일들과 텍스트 프롬프트 입력을 사용해서 
    Gemini API의 gemini-2.5-flash-image-preview 모델을 기반으로 이미지 생성 
    (API 사용 예시 : @example.py , API Key는 .env의 GEMINI_API_KEY 에서 불러오도록 함)
4. 생성한 이미지를 output 폴더에 만들어주고 다운로드로 가능하게 해줘.  

웹서비스는 Flask, Jinja2 패키지를 사용해서 개발해줘.
서비스 명 : 나도 바나나, 매인 컬러 : #FFD548, 로고 : logo.png 
위 내용을 기반으로 프로그램을 만들어줘.
